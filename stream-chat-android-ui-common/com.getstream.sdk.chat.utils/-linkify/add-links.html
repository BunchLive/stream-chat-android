fun addLinks(textView: TextView)
Content copied to clipboard

Scans the provided TextView and turns all occurrences of the link types into clickable links (Currently only support web urls). If matches are found the movement method for the TextView is set to LinkMovementMethod.

NOTE: Because this implementation doesn't remove existing URLSpan, make sure it is not repeatedly called on same text.

Parameters

textView

TextView whose text is to be marked-up with links.
